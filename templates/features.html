Capabilities

Core tools for safe database operations.

PromptDB combines AI command generation, validation, audit history, and manual data management.

AI

Natural Language Parser

Uses OpenRouter to transform plain English into strict JSON commands before any database operation runs.

DB

MongoDB CRUD Layer

Supports controlled find, insert, update, and delete operations against the users collection.

UX

Preview and Confirmation

Shows the generated MongoDB command and requires confirmation before write operations run.

OPS

Audit Logging

Tracks prompts, commands, parser source, action status, and result summaries for operational visibility.

SAFE

Admin Access Control

Protects the console, users dashboard, exports, logs, and database APIs behind a Flask session login.

STATE

Users Dashboard

Provides manual CRUD, search, sorting, and CSV or JSON export for the users collection.
